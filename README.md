## ✨ Features

- 🎨 **Modern Gradient UI** — تصميم عصري بتدرجات لونية جذابة
- 📂 **Course Category Tabs** — تصفية الدورات حسب الفئة (Frontend / Backend / Mobile / Data)
- 🔢 **Animated Counters** — عدادات متحركة للإحصائيات
- ❓ **FAQ Accordion** — أسئلة شائعة تفاعلية
- 💬 **Testimonials** — آراء الطلاب مع بطاقات جذابة
- 💰 **Pricing Plans** — خطط اشتراك مع تمييز الخطة المميزة
- 📱 **Fully Responsive** — متوافق مع جميع الأجهزة
- 🍔 **Mobile Navigation** — قائمة جانبية للشاشات الصغيرة
